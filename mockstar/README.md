# mockstar 的学习项目

本项目为 [mockstar](https://github.com/mockstarjs/mockstar) 的学习项目，我们整理了最常使用的场景，分多个学习主题，每个学习主题所包含的技能点在 README.md 中均描述，建议 clone 项目之后亲自实践！
