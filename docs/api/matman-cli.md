## matman-cli


| 命令  |       选项        |                     说明                     |
| :---: | :---------------: | :------------------------------------------: |
|   /   | --version、--help |          `查看版本`、`显示帮助信息`          |
| build |   -d 或者 --dev   |   `编译爬虫脚本`，-d 与 --dev 生成调试版本   |
| init  |  -t 或者 --type   | `初始化项目`，-t 与--type 指定生成所用的模板 |
