# 03. 更多例子

我们为大家提供了一个 学习项目，我们整理了最常使用的场景，分多个学习主题，每个学习主题所包含的技能点在 README.md 中均描述，建议 clone 项目之后亲自实践！

  • 01.getting-started: 最简单的示例，适合入门
  • 02.individual-crawler-script: 独立的爬虫脚本
  • 03.crawler-inject-jquery: 注入 jQuery 方便写爬虫脚本
  • 04.device-emulation: 设备仿真，定义无头浏览器ua和大小
  • 05.launch-options: 浏览器启动参数，定义展现模式，记录器等
  • 06.set-cookie: 设置浏览器 Cookie 的各种方式
  • 07.set-screen-shot: 设置截屏的各种方式
  • 08.result-coverage: 设置截屏的各种方式
  • 09.matman-config: matman 的配置项目
  • 10.mockstar-and-proxy: matman 设置代理与 mockstar 指定返回
  • 51.e2e-test: 使用 matman 做端对端测试的简单例子
